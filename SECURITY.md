# Security Policy

Thanks for helping keep Newdich Store secure. If you discover a vulnerability, please report it privately.

## Supported Versions
Only the latest version on the main branch is supported.

## Reporting a Vulnerability
Please send a report with:
- A clear description of the issue
- Steps to reproduce
- Impact assessment
- Any proof of concept (if safe to share)

Send reports to the project owner or the security contact for your organization. If you do not have a security contact, open a private issue or reach out to the repository owner directly.

## Disclosure Process
We will:
- Acknowledge receipt within 3 business days
- Provide a remediation plan or timeline when confirmed
- Credit reporters on request after a fix is released

## Scope
In scope:
- Public storefront
- Admin dashboard
- API integration points

Out of scope:
- Issues requiring physical access
- Social engineering attacks
- Denial of service testing that impacts production users
